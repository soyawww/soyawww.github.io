主流浏览器：拥有shell（外壳【展示页面】）和内核。(市面上的很多浏览器只有shell)

例如：IE（trident），Firefox（Gecko），google chrome（Webkit/blink），safari（webkit），opera（presto）


web前端=html（结构）+css（样式）+JavaScript（行为）

在开发时，通常各尽其职。即便有另一个职务的功能，也不做编写使用。


一、HTML基础

1.html的基本结构

  • 根标签（只有一种）：<html></html>

    结构化标签：例如<head></head>,<body></body>


  • Doctype（全称document type，文本声明，声明格式【例如html，xhtml】。）

    HTML 4时有很多种模式，例如严格模式，混杂模式（又称怪异模式）。

    如果没有声明模式，就会默认为怪异模式。怪异模式会影响css的呈现，不同浏览器间差异较大。所以需要兼容。

    HTML5就统一标准了，只需要兼容pc端，移动端间的内容。


2.标签分类

  1. 行级元素（内联元素 inline）

    特点：内容决定元素所占位置，不可以通过css改变宽高。

    例如：<span> <em> <strong> <a> <del>

  2. 块级元素（block）

    特点：独占一行，可以通过css改变宽高

    例如：<div> <p> <ul> <li> <ol> <form> <address>

  3. 行块级元素（特殊） inline-block

    特点：内容决定大小，可以改宽高

    例如：<image>


注：标签自带display元素，默认为各自标签的分类。

​ 所以可以通过更改display属性来更改它们的样式。

​ 凡是带有inline的元素都含有文字特性，都叫文本类元素。

​ （自带padding和margin）

​ 行级元素只能嵌套行级元素，块级元素可以嵌套行级和块级。

​ （所以<p>标签不可以嵌套<div>）


3.基础标签

<meta>可提供有关页面的元信息（meta-information）

<meta charset='utf-8'> （charset：字符集）
<meta content='服装'>

（【ps】常用编码字符集：

1.GB2312 国标第2312条 也叫《信息交换用汉字编码字符集》（缺点：只能识别简体中文集）

2.gbk GB2312的扩展，拥有繁体

3.Unicode 万国码

4.utf-8 unicode的8比特升级版）


<title>标题

<p>段落标签（使内容成段）

<h1——h6> 1-6级标题

<strong> 加粗

<em>斜体

（【ps】标签可以嵌套，加粗斜体：

）

<delete>划线效果

<address>地址（成段展示+斜体）

<div>和<span>容器（分块，让页面更加结构化，使整块操作）

容器的特点：1.结构化 2.绑定化操作

（【ps】空格和回车为文本分隔符，让浏览器识别是不同的单词；

 等于一个空格的表现形式，为html编码。

常用html编码：

  1. $nbsp;（空格）
  2. &lt;（less than缩写，小于符）
  3. &gt;（great than缩写，大于符））

<br>回车，单标签（例如mate，hr【水平线】）


4.高级标签

（1）<ol>和<li>有序列表

效果：

<ol>属性

ol的type默认为1（<ol type="1">）

ol的type的值对应的效果（一共有五种）：

1（按照阿拉巴数字排序）

a（按照小写字母a，b，c排序，多余26个字母开始26进制，例如aa，ab）

A（按照大写字母A,B,C排序，多余26个字母开始26进制，例如AA，AB）

i（按照小写罗马数字排序）

I（按照大写罗马数字排序）

reversed=”reversed” 倒叙

start 从第几个开始排序。（例如start=”3”，从3开始排序）


（2）<ol>和<li>无序列表

效果：


<ul>属性

仅有一个属性type。

默认为type=”disc”实心圆。

circle 空心圆

square 实心方块


tips：ol（ul）与li为父子结构

一个大的功能由很多功能子项组成，每一个功能子项除了内容不同，它的功能样式基本相同。

这样的功能适合使用父子结构。

（3）<img>图片标签

单标签。

src 资源，引入图片

（地址分为三类：1.网上url 2.本地的绝对路径 3.本地的相对路径）

alt 图片占位符，当图片出错时，展示文字信息。

title 图片提示符。当鼠标移动在图片上时，提示用户信息。


（4）<a>标签

anchor 锚

功能：1.超链接 2.锚点 3.打电话 4.协议限定符

herf 超文本应用（协议也需要写上，例如herf=”https://www.baidu.com")

特殊：tele：打电话（移动端用，例如<a href=’’tele:13530xxxxxx’>）

mailto：发邮件（同上）

协议限定符：例如herf=”javascript:while(1){alter(‘点’)}”,限定javascript。

target 规定在何处打开链接文档

（例如target=”_blank”，可在新的标签页打开要跳转的目标。

target几个属性值：

  1. _self 默认，在相同框架打开
  2. _blank 在新窗口中打开被链接文档
  3. _parent在父框架集中打开被链接文档
  4. _top在整个窗口中打开被链接文档）

<a>标签的内容可以是任何形式的，例如图片，css


（5）<form>表单标签

可以发送数据给后端。

method 发送数据的方式，有两种：GET和POST

发送数据一定要有数据值和数据名，否则无法提交成功。

action 发送目标（目的地，后端位置）


（6）**<input>标签**

type决定类型（text 输入框，password 密码框，submit提交，radio单选框，checkbox复选框）

name数据名。

onfocus 聚焦，inblur失去焦点。

type的为radio/checkbox时，name相同的为同一道选择题。

**checked=”checked”**让选择默认选中，减少了用户的操作，增强体验。


（7）<select>与<option>下拉菜单

和ol，li一样是父子结构。

option标签就赋予了value值，不需要单独添加。

若赋予了value值，则以value值为标准。

二、css基础

css无法单独存在，必须有html。


css全称cascading style sheet（层叠样式表）

css是浏览器下载一行执行一行。

可以先写css（先定义功能），后写html


注释：/* */ （css只有这一种注释方式）

1.引入css

有不同的方式引入css

  1. 行间样式（直接在标签引用）
  2. 页面级css（在head中：<style type="text/css"></style>）
  3. 外部css文件（使用link标签，连接到css）

2.基础选择器

（1）id选择器

id=”demo” #demo

一对一，一个元素只能有一个id值，一个id只能对一个元素。

（2）class选择器

class=”demo” .demo

多对多，一个class可以对应多个元素

例如：class=”demo1 demo2”用空格就可对应两个class

（3）标签选择器

例如div，span

无论嵌套多少层，都可以应用

（可以利用标签选择器，初始化标签）

（4）通配符选择器

* 表所有

（可以利用通配符选择器，初始化所有标签）

（5）属性选择器

[ ] 可以根据元素的属性及属性值来选择元素。

例如[id]，选中所有的id


★权重

优先级：**!important >行间样式> id > class | 属性 > 标签 >** 通配符

css权重值
!important Infinity（正无穷）
行间样式 1000
id 100
class|属性|伪类 10
标签|伪元素 1
通配符 0

tips:

  1. Infinity在计算机中可计算，所以Infinity+1>Infinity。
  2. css权重值为256进制
  3. 写在同一排的选择器，权重值相加（权重相同，取后面的）

3.复杂选择器

（1）父子选择器/派生选择器

使用空格分离。

每一个层级并不一定需要同样的选择器。

例如(例子1): .wrapper .box em{background-color:red;}

浏览器是从右往左找，也就是例子中的em→.box→.wrapper

（2）直接子元素选择器

使用**>**。

例如(例子2)：div>em{background-color:red;}

div下直接一级的em

（3）并列选择器

用多个限制条件选中一个元素，并且不加空格。

例如(例子2)：em.box{}

注：顺序不可写反，组合使用时标签选择器放在前面

（4）分组选择器

当多个对象定义了相同的样式时，就可以把它们分成一组，共用一个代码块，减少冗余。

效果：

（5）伪类选择器

伪类：同一个标签，根据其状态不同而呈现不同样式。用冒号**:**表示。

伪类分为静态伪类和动态伪类两种。

  • 静态伪类：只能作用于超链接。

:link 超链接点击之前

:visited 链接被访问之前

  • 动态伪类：针对所有标签都适用的样式。

:hover 用于选择鼠标指针浮动在上面的元素

:active 用于鼠标点击标签时

:focus 某个标签获得焦点时


<a>标签的四种伪类：link，visited，hover，active

（ps：a:hover 必须被置于 a:link 和 a:visited 之后，才是有效的，a:active 必须被置于 a:hover 之后，才是有效的。）


（6）伪元素选择器

伪元素：并不是真正的页面元素，可以当作元素操作，但是没有元素结构，无法在html中审查。

​ 都必须设置content，内容由content控制。

​ 它是行级元素，用**双冒号::**表示。

::after :在子元素前增添一个伪元素。

::before ：在子元素后可以增添一个伪元素。


4.css代码块属性

font-size：字体大小（默认为16px）【设置的是字体的高】

font-weight：默认值为normal，其他几个值：bold（加粗）lighter（细体）boder（更粗）；也可以用数字<integer>100——900来表示【strong标签的效果，自带bold】。

font-style：定义字体的风格。默认值normal（标准），其他值：italic（斜体）【和em标签作用相同】，oblique（倾斜）

font-family：字体。通用字体为arial（乔布斯发明）

字体颜色有三种写法：

  1. 纯英文单词(red,green)

  2. 颜色代码（#ff4400）

    光学三原色：r g b （红绿蓝）

    颜色代码由三原色组成，每两位代表一个颜色，且每两位是一个十六进制数（00-ff）。

    第一组十六进制数代表红色的饱和程度，第二组代表绿色的，第三组代表蓝色的。

    所以#000000：黑色，#ff0000：红色，#00ff00：绿色，#0000ff：蓝色。

    如果颜色代码每两位都是重复的，可简写（例如ff4400→f40）。

  3. 颜色函数

    rgb(255,255,255)

    括号内的值是0-255

    tips:transparent为透明色

border：

给容器加一个盒子（外边框）。它是一个复合属性。

例如border:1px solid black;

这三个属性依次为

border-width边框粗细，

border-style边框样式（solid实体，dotted虚点状线，dashed条状虚线），

border-color边框颜色

每个边可以单独设置，border-left，border-top，border-right，border-down

例如设置上边为红色：border-top-color:red;



width：宽

height：高

text-align：对其方式。默认为left，居中为center，向右为left。

line-height：单行文本所在的高度。(若想让垂直文本居中，就设置line-height的值等于容器高度）

text-indent：首行缩进（值为2em为缩进两文本【1em=1font-size=16px】）

text-decoration****：文本装饰。none无线，underline下划线，line-through【效果和<del>相同，但用时选择前者】，overline上划线

cursor：光标。pointer光标呈现为指示链接的指针（一只手）【值很多，无需记】

opacity：透明度


z-index：层级。


float：浮动。可以使元素站队（站队的边界是父级的边界）。还可以在浮动的同时，加margin。

浮动元素会产生浮动流（并不是分层）：所有产生了浮动流的元素，块级元素看不到他们。

产生了bfc的元素和文本类属性的元素以及文本都能看到浮动元素。

clear：清除周边浮动流。值有right，left和both。

（只能在块级元素上生效）

如果想利用伪元素清除浮动，必须先转换为块级元素（display: block;)

如果设置了position:absolute; 或float:left/right;

系统内部会自动把元素转换成inline-block


backgroud:背景

linear-gradient() 函数用于创建一个线性渐变的 “图像”。

background-image：url(xxx.jpg);背景图片

background-size:100px 100px 背景图片大小

background-repeat:背景图片平铺方式 （值：no-repeat【不重复】，repeat-x（x轴重复，横向重复），repeat-y（y轴重复）

background-position:100 100背景图片位置（可以填数字，百分数，也可以填left、top、center等）

1）position：absolute；

2）display：inline-block；

三、盒子模型

盒子的三大部分：

盒子的组成部分：盒子壁 border，内边距 padding，盒子内容 width+height

盒子模型：margin+border+padding+（content=width+height）

关于取值的顺序问题：

四个值时，顺序为上右下左

例如：padding：100px 150px 200px 250px

上100，右150，下200，左250

三个值时，顺序为上左(右)下

层模型

position：定位

left、right、top、bottom只和position配合使用。

上面四项均为其方向的边线对浏览器边线的距离。

position属性

absolute：脱离原来位置进行定位。当一个元素成为absolute时，它就脱离原来的层跑到了上一个层，它原来的层就真空了。

相对于最近的有定位的父级进行定位，如果没有，相对于文档进行定位。

relative：保留原来位置进行定位。

相对于原来的位置进行定位。

一般用relative作为参照物，而使用absolute进行定位。

fixed:固定定位。（滚动条怎么动，它都不动）

实例：居中五环，两栏布局

居中五环

关键点：画圆，层级

两栏布局

关键点：一个元素让一块位置，一个元素定位到被让的位置上去
